## 🚀 EXACT STEPS (Copy-Paste करें!)

### STEP 1: PowerShell खोलो
```powershell

cd "c:\Users\Nipun\Desktop\AI-Based-Fault-Detection-in-Transmission-Infrastructure"
```

### STEP 2: Git Install किया है? Check करो
```powershell
git --version
```

**अगर error आए:** https://git-scm.com/download/win से install करो

---

### STEP 3: Git Configure करो
```powershell
# अपना नाम और email डालो:
git config --global user.name "Nipun"
git config --global user.email "your.email@gmail.com"
```

---

### STEP 4: Code को GitHub में Push करो

**Option A: अगर पहली बार है:**
```powershell
git init
git add .
git commit -m "Transmission Fault Detection - Ready"
```

**Option B: अगर पहले से हो:**
```powershell
git add .
git commit -m "Update"
```

---

### STEP 5: GitHub में Repository बनाओ

1. जाओ: https://github.com/new
2. Fill करो:
   - Name: `transmission-fault-detection`
   - Visibility: `Public` select करो
3. "Create repository" click करो
4. GitHub तुम्हें commands दिखाएगा - उन्हें copy करो!

---

### STEP 6: GitHub Commands चलाओ
```powershell
# GitHub ने जो commands दिए, वो paste करो:
# (अपना username डालना - नीचे का example है)

git remote add origin https://github.com/YOUR-USERNAME/transmission-fault-detection.git

git branch -M main

git push -u origin main
```

---

### STEP 7: GitHub Account बनाओ (अगर नहीं है)

1. जाओ: https://github.com/signup
2. Email डालो
3. Password डालो
4. Username चुनो
5. Email verify करो

---

### STEP 8: Streamlit Cloud में Deploy करो

1. जाओ: https://share.streamlit.io
2. "Sign up with GitHub" करो
3. "New app" click करो
4. Fill करो:
   - Repository: `YOUR-USERNAME/transmission-fault-detection`
   - Branch: `main`
   - Main file: `app/main.py`
5. "Deploy" click करो
6. 3-5 मिनट इंतजार करो ⏳

---

### STEP 9: ✅ Done! 

तुम्हारा URL मिलेगा:
```
https://transmission-fault-detection-YOUR-USERNAME.streamlit.app
```
